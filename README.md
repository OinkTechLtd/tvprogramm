# 📺 TV-CHECKPROGRAMM

Полноценный сайт-агрегатор программы телепередач и радиоэфира.  
Аналог `programma-peredach.com` с современным тёмным UI и подгрузкой реального EPG (XMLTV) с публичных источников.

---

## 🚀 Быстрый старт

Просто откройте `index.html` в любом браузере — никаких зависимостей, серверов или сборки не требуется.

```
tv-checkprogramm/
├── index.html         # Главная точка входа
├── css/
│   └── style.css      # Все стили (CSS переменные, адаптив, анимации)
├── js/
│   ├── data.js        # База каналов + генератор расписания
│   └── main.js        # SPA-логика, рендеринг, события
└── README.md
```

## 📝 Расширение

Чтобы добавить новый канал, в `js/data.js` в массив `CHANNELS` добавьте:

```js
{ 
  id: 'my-channel',           // уникальный ID
  name: 'Мой Канал',          // отображаемое название
  type: 'tv',                 // 'tv' или 'radio'
  category: 'entertainment',  // категория (см. CATEGORY_LABELS)
  icon: 'МК',                 // 1-3 символа для иконки
  color: '#E53935'            // цвет акцента
}
```

В текущей версии `getSchedule()` сначала возвращает fallback-расписание, затем асинхронно подменяет его на реальное XMLTV-расписание (если источник доступен).
